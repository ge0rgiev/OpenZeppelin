# OpenZeppelin Example Usages

Install dependencies:
```shell
npm i
```

Hardhat shorthand (Optional) hh -> npx hardhat:
```shell
npm i -g hardhat-shorthand
```

Create an enviroment file named `.env` and use `.env.example` as template.

Run the test suites:

```shell
hh test
```
