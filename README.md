# OpenZeppelin Example Usages

Install dependencies:
```shell
$ npm i
```

Hardhat shorthand (Optional) hh -> npx hardhat:
```shell
$ npm i -g hardhat-shorthand
```

Run the test suites:

```shell
$ hh test test/utils/structs/enumerable.set.test.ts
```
